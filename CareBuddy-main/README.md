# CareBuddy(Medication Reminder App)
This is our team project. We are developing the CareBuddy System. 
The project is about helping users manage their medication schedules. In real life, many patients forget to take medicines on time, which can lead to serious health problems. This system solves that issue
by allowing users to input their medications and automatically receive reminders.From a software engineering perspective, the application is designed using modern Android development practices, ensuring that the system is not only functional but also
maintainable, scalable, and easy to extend in the future.
The system is very user friendly .
The project will be developed by all our member.
The Front end, backend and UI/UX design is separetly designed by each member.

<b>  📦 CareBuddy </b>
<p> CareBuddy application provides a comprehensive and practical solution
for medication management. With potential future enhancements such as cloud
synchronization, multi-user support, and intelligent reminder systems, the application can
evolve into a more advanced healthcare assistant, making it highly valuable for real-world
use.</p>
<b>🚀 Features</b>
<ul>
  <li>✅Add medication details (name, dosage, timing)</li>
  <li>✅ Set schedules (daily, weekly, etc.) </li>
  <li>✅ Receive notifications at the correct time </li>
  <li>✅ Track whether the medicine was taken or missed</li>
</ul></br>
